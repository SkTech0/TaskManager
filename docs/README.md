## Task Management System

This repository contains a production-grade Task Management System implementing a clean, layered architecture with a .NET 8 backend, Angular 18 SPA frontend, PostgreSQL database, and Docker-based deployment.

### Features

- User registration and login with JWT authentication
- Secure task CRUD operations
- Search across title, description, remarks, and status
- Audit fields including created/updated timestamps and users
- Clean application layering and separation of concerns
- Automatic database migrations and seed data on startup
- Dockerized backend, frontend, and PostgreSQL

Refer to `architecture.md`, `db-design.md`, `api-docs.md`, and `build-run.md` for detailed documentation.
